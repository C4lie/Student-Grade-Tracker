# Student Grade Tracker

## How to Run

1. Open your terminal and navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to the URL provided in the terminal (e.g. `http://localhost:5555`).
